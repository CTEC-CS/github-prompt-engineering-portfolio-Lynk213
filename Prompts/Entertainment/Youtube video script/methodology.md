# Design Justification
- I used the RTF framework. It seemed the simplest and most understandable for this prompt.
- I put the age part of the prompt in because I wanted to let a user of this framework know that they can use it no matter how old they are.
- I included a tone indicator because I thought it would help the user assess their target audience.
- This prompt effectively guides the AI because it puts the AI into a plausible situation where it needs to use its near infinite database of information and create a video on a specific topic. It provides a good, mostly specific framework and way of carrying it out.
- I required the user to input quite a few things. I don’t necessarily want people to put minimal thought into their creative works.
### Best use cases
- When you don’t have time to research and write a script about something you’re passionate about.
- As an example video script when you run out of ideas and have to write a video essay script.
- When you ran out of time for an assignment in your video making class that tests on techniques and not scriptwriting.
